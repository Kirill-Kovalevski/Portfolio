Git Rescue Kit — fixes for the top 10 “oh no” moments
  • Undo last commit, keep changes: git reset --soft HEAD~1
  • Fix message: git commit --amend
  • Discard local edits: git restore .
  • Restore from origin/main: git restore -s origin/main path/to/file
  • Stash & return: git stash -u → git stash pop
  • Squash last N: git reset --soft HEAD~N && git commit -m "one"
  • Bring back a branch: git reflog → git checkout -b branch SHA
  • Delete remote: git push origin --delete branch
  • Align exactly: git switch main → git reset --hard origin/main
  • Refresh default HEAD: git remote set-head origin -a
Debugging Without Tears: 7 Rituals That Actually Work
  1. Reproduce: lock versions; write exact 3 steps.

  5. Repair: add a test; prevent regressions.
  6. Review: one hypothesis per change.

TypeScript Patterns that Save Hours 
// Narrowing helper
export const isNonNull = <T,>(v: T | null | undefined): v is T => v != null;

// Result type
type Result<T,E=Error> = { ok:true; value:T } | { ok:false; error:E };

// Safe fetch
async function getJSON<T>(u:string):Promise<Result<T>>{
  try { return { ok:true, value: await (await fetch(u)).json() as T }; }
  catch(e){ return { ok:false, error:e as Error }; }
}
  • Prefer Result<T> to throwing in app code.
  • Use Pick, Partial, ReturnType.
  • From JSON → TS: VS Code “Copy as TypeScript”.
